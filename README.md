# VoltEdge RevOps Automation Demo

Portfolio demo showing HubSpot, n8n, CRM data quality, lead/product sync, Stripe-style revenue sync, ad conversion feedback, validation, dedupe controls, association logic, reconciliation queues and reporting outputs.

## Positioning

This repo is designed as a portfolio/interview evidence pack for RevOps, Marketing Ops and automation work.

It demonstrates the control layer around CRM and marketing automation workflows:

- Source data validation
- Required field checks
- HubSpot property mapping
- Company/contact dedupe logic
- Contact/company association logic
- Revenue sync logic
- Lifecycle and revenue conversion feedback
- Conversion event dedupe controls
- Recommendation/reporting outputs
- Error and reconciliation queues
- Guardrails for production implementation
