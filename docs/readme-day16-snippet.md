## Day 16 - API Deal and Association Logic

Day 16 demonstrates mock HubSpot deal upsert and association planning.

It includes:

- opportunity input data
- deal create/update logic
- contact-company association logic
- deal-company association logic
- deal-contact association logic
- lifecycle/deal stage rules
- duplicate deal prevention
- validation and generated logs

Run locally:

```bash
node scripts/day16-deal-association-logic.js
```

Review output:

```bash
cat logs/day16-generated-association-log.json
```
