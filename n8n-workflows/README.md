## Recommended export process

1. Open the workflow in n8n.
2. Click the three-dot menu.
3. Choose `Download` or `Export`.
4. Save the JSON file into this folder.
5. Commit the file to GitHub.

- `day-12-stripe-revenue-sync-demo.json`
